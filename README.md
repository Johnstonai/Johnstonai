## What's Built Into This Page

### On-Page SEO
- Optimized `<title>` tag and meta description
- Canonical tag
- Open Graph tags for social sharing
- `robots` meta tag
- H1 → H2 → H3 → H4 content hierarchy
- Keyword-rich copy targeting Fresno, Central Valley, and all service lines
- Local entity signals woven throughout (city names, license numbers, service area)

### Schema Markup (JSON-LD)
| Schema Type | Purpose |
|---|---|
| `HVACBusiness` | Business identity, address, phone, hours, geo coordinates |
| `FAQPage` | Direct answer targeting for AI Overview and voice search |
| `AggregateRating` | Star rating signals for rich results |
| `OfferCatalog` | Service inventory for AI parsers |
| `BreadcrumbList` | Site structure signals |
| `PostalAddress` + `GeoCoordinates` | Local map pack eligibility |
| `areaServed` | City-level service area declaration |

### AEO / GEO Layer (The AI Visibility Differentiator)
- 6-question FAQ section structured to match how people ask AI assistants
- Direct-answer format — no fluff, no buried answers
- Entity clarity: business name, location, license, services stated plainly
- Questions target high-intent Fresno/Central Valley search queries

### Technical
- Pure HTML/CSS/JS — zero dependencies, zero frameworks
- Mobile responsive
- Fast-loading (no bloat, no external scripts except Google Fonts)
- Accessible — ARIA labels, semantic HTML, keyboard-navigable FAQ
- Single file — drop and deploy anywhere
